SOLID WASTE COMMERCIAL CONTAINER OPERATION DASHBOARD (SWCCOD)

 
 


 

 

 

The Electronic Dashboard replaces the white board. The Electronic Dashboard can be displayed on multiple monitors within the Solid Waste department. Changes and updates are made using a new program called the Manager. The Manager has the capability to make updates from any computer within the City.

 

 

Route Manager

 

 

 

 

 

 

 

The Route Manager manages the display of the Electronic Dashboard. All display changes are made using the Manager. The change automatically displays on the Electronic Dashboard. Security levels are available to any part of Electronic Solid Waste Operation Dashboard to limit who has access to which features.

Administrative Manager

 

 

 

 

 

 

 

 

 

 

 

 

Administrative manager manages commissioning and decommissioning of operators, call signs, vehicle numbers and user access.

 

 

 

 

 

 

 

Vehicle number and call sign administrator. Allows you to commission and decommission vehicle numbers and call signs.

 

Allows assignment of user authorization levels. There are four authorization levels:

1.   None user has no authorization.

2.   Read only users can view dashboard and dashboard manager.

3.   Editor user can edit dashboard manager and container inventory.

4.   Administrator all above roles plus can assign access and authorization levels.

 

 

 

 

 

 

 

 

 

 

 

 


 

 

 

 

 

 

 


 

 

 

 

 

 

 

 

 

Container Inventory

 

 

 

 

 

 

 

 

 

 

 

Container Inventory page tracks available containers for delivery. It tracks containers that are on hand but currently not assigned to a customer. Also allows notations regarding each container size. Notations are immediately reflected on the main dashboard screen.
